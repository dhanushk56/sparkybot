Frequently Asked Questions

Quick answers to the most common questions about SparkyBot.

Is SparkyBot really free?

Yes! SparkyBot is 100% free forever. No paywalls, no premium tiers, no nitro‑gating. All features are available to every server.

What permissions does SparkyBot need?

SparkyBot requires standard moderation permissions: Kick Members, Ban Members, Manage Channels, Manage Roles, Send Messages, Embed Links, Attach Files, Connect & Speak (for music), and Moderate Members. You can customize permissions during invite.

How do I change the bot's prefix?

Use the /prefix set slash command or the $prefix set command. Server admins can set a custom prefix anytime.

Can I self‑host SparkyBot?

Absolutely! SparkyBot is open‑source and fully documented. Check the GitHub repository for installation instructions.

How do I report a bug or suggest a feature?

Join our support server and use the /rp bug or /rp suggest commands. You can also open an issue on GitHub.

Is my data safe?

Yes. SparkyBot only stores minimal configuration data (server settings, economy balances, etc.) and never shares it with third parties. You can reset any module's data at any time.
